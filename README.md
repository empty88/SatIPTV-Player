# SatIPTV-Player

## Features

- eigene Senderliste zusammenstellen
- Finden von SAT>IP-Servern im Netzwerk
- Senderauswahl von Fritz!Box Cable (DVB-C) oder M3U-Liste (DVB-S)
- Kanallogos (picons)
- EPG (Tvheadend benötigt)
